# The Fireplace — Icon Assets

## Overview

The Fireplace uses a custom flame icon across all platforms (macOS, iOS, web). The icon features a warm amber/orange gradient that reflects the "fireplace" theme and works well in both light and dark contexts.

## Design Philosophy

- **Simple silhouette**: Works at all sizes from 16x16 menu bar to 1024x1024 App Store
- **Warm palette**: Amber (#fbbf24) to orange (#f59e0b) to red (#dc2626) gradient
- **Dark mode optimized**: Designed for The Fireplace's dark-only interface
- **Recognizable**: Flame shape is instantly identifiable

## Icon Files

### Source Files

Located in `/src-tauri/icons/`:

- **flame.svg** — Master icon (1024x1024) with full gradient
- **tray-template.svg** — Menu bar template (32x32, black with alpha)

### Generated Assets

#### macOS App Icons
- `icon.icns` — Multi-resolution icon bundle for macOS
- `icon-1024.png` — 1024x1024 (App Store, Finder)
- `icon-512.png` — 512x512
- `icon-256.png` — 256x256
- `128x128@2x.png` — 256x256 retina
- `128x128.png` — 128x128
- `32x32@2x.png` — 64x64 retina
- `32x32.png` — 32x32
- `16x16.png` — 16x16

#### Menu Bar Icons (Template Images)
- `tray-icon@2x.png` — 32x32 retina (black with alpha)
- `tray-icon.png` — 16x16 (black with alpha)

Template images render in the system accent color and support dark mode automatically.

#### Windows
- `icon.ico` — Multi-resolution ICO (16, 32, 256)

#### Web/Favicon
- `/public/favicon.png` — 32x32 PNG
- `/public/favicon.ico` — Multi-size ICO

## Regenerating Icons

If you need to modify the icons:

1. Edit the source SVG files in `/src-tauri/icons/`
2. Run the regeneration script:

```bash
cd src-tauri/icons

# Regenerate all sizes from flame.svg
rsvg-convert -w 1024 -h 1024 flame.svg > icon-1024.png
rsvg-convert -w 512 -h 512 flame.svg > icon-512.png
rsvg-convert -w 256 -h 256 flame.svg > icon-256.png
rsvg-convert -w 256 -h 256 flame.svg > 128x128@2x.png
rsvg-convert -w 128 -h 128 flame.svg > 128x128.png
rsvg-convert -w 64 -h 64 flame.svg > 32x32@2x.png
rsvg-convert -w 32 -h 32 flame.svg > 32x32.png
rsvg-convert -w 16 -h 16 flame.svg > 16x16.png
cp icon-256.png icon.png

# Regenerate tray icons from tray-template.svg
rsvg-convert -w 32 -h 32 tray-template.svg > tray-icon@2x.png
rsvg-convert -w 16 -h 16 tray-template.svg > tray-icon.png

# Create macOS .icns bundle
mkdir -p icon.iconset
cp 16x16.png icon.iconset/icon_16x16.png
cp 32x32.png icon.iconset/icon_16x16@2x.png
cp 32x32.png icon.iconset/icon_32x32.png
cp 32x32@2x.png icon.iconset/icon_32x32@2x.png
cp 128x128.png icon.iconset/icon_128x128.png
cp 128x128@2x.png icon.iconset/icon_128x128@2x.png
cp icon-256.png icon.iconset/icon_256x256.png
cp icon-512.png icon.iconset/icon_256x256@2x.png
cp icon-512.png icon.iconset/icon_512x512.png
cp icon-1024.png icon.iconset/icon_512x512@2x.png
iconutil -c icns icon.iconset -o icon.icns
rm -rf icon.iconset

# Copy to web public directory
cp favicon-32x32.png ../../public/favicon.png
cp icon.ico ../../public/favicon.ico
```

## Requirements

- **librsvg**: `brew install librsvg` (provides `rsvg-convert`)
- **iconutil**: Built-in macOS tool for .icns generation

## Color Palette

The flame icon uses these Tailwind colors:

- Outer flame: `amber-400` → `amber-500` → `red-600`
- Inner flame: `amber-50` → `amber-400` → `amber-500`
- Hotspot core: `amber-50` with radial fade

These align with The Fireplace's design system (zinc/amber).

## Platform Requirements

### macOS
- **App Icon**: 1024x1024 PNG or .icns bundle
- **Menu Bar**: 16x16 and 32x32 template PNGs (black with alpha)
- **Template rendering**: Icons automatically adapt to menu bar theme

### iOS
- **App Icon**: Multiple sizes from 20x20 to 1024x1024
- **Asset catalog**: Managed by Xcode in `src-tauri/gen/apple/`
- **No transparency**: iOS app icons must be opaque

### Windows
- **App Icon**: .ico file with 16, 32, 48, 256 sizes
- **Transparency**: Supported

## Testing

To verify icons render correctly:

```bash
# Build and run the app
pnpm tauri build

# Check menu bar icon in macOS
# Should render in system accent color and support dark mode
```

## Notes

- Menu bar icons MUST be template images (single color with alpha)
- Template images use `iconAsTemplate: true` in `tauri.conf.json`
- iOS icons are auto-generated by Tauri from the PNG assets
- Always maintain the SVG source files — never edit PNGs directly
